# Перенос проекта с Vite + React на Next.js

## Правки по прошлому дз:
1. Разнес верстку по компанентам, смогу реализовать многокрастное использование компонентов
2. Компоненты больше не хранят в себе логику кроме загрузки сторов, вся логика перенесена в сторы
3. Каждой странице соответствует свой локальный стор
4. Реализовал все интерфейсы, провайдеры и контексты, но при внедрении createLocalContext в next.js появились сложности, будет доработанно в будущем
5. Доработал визуал некоторых страниц

## Что сделано в этом ДЗ:
1. Реализована структура App-Router, error и not-found страницы сделаны
2. Реализовал логику pre-fetch для страницы со всеми фильмами и для каждой страницы по-отдельности
3. Есть пример параллельной загрузки, настроены кеширование и ревалидация
4. Создан клиентский провайдер сторов, все компоненты-наблюдатели отмечены 'use client'
5. Метаданные и изображения настроены
