## Проект на Next.js с использованием Mobx, TypeScript и Frame.Motion##
О проекте
Современное веб-приложение для просмотра фильмов с авторизацией, избранным и плавными анимациями. Построено на Next.js 14+ с использованием TypeScript, MobX для управления состоянием и Framer Motion для анимаций.

🚀 Технологии
Next.js 14 (App Router)

TypeScript

MobX + MobX React Lite

Framer Motion

CSS Modules / TailwindCSS

LocalStorage / JWT (для авторизации)

📦 Функционал
🔐 Личный кабинет
Регистрация и авторизация пользователей

JWT токены с refresh-механизмом

Защищённые маршруты (PrivateRoute)

Просмотр и редактирование профиля

🎥 Страница фильмов
Список фильмов с пагинацией/бесконечной прокруткой

Фильтрация по жанрам, году, рейтингу

Поиск по названию

Детальная страница фильма

❤️ Избранное
Добавление/удаление фильмов в избранное

Отдельная страница с избранными фильмами

Сохранение избранного в MobX store и синхронизация с API

Персистентность (сохранение в localStorage)
